# VERBA Executive Review — Codex instructions

## Goal
Build a premium digital editorial executive review for VERBA Medical Center. The primary output is an HTML experience for GitHub Pages, not a PowerPoint deck.

## Source of truth
- Read `source/quarterly-report-q1-2026.md` for Q1 2026 facts and management conclusions.
- Unpack and read `source/presentation-skills-public-pack.zip` before changing the visual product.
- Use the existing published project as the technical and visual starting point, but do not merely replace Q2 text with Q1 text.
- Use official VERBA MAYR photography from `https://verbamayr.ru/` as the primary image source.

## Required workflow
1. Apply the story-architecture skill to define the management narrative.
2. Interpret the current site and official VERBA materials into a visual direction.
3. Define a reusable Visual DNA.
4. Build the result as a digital editorial product.

## Visual priority
The result must feel like premium medical intelligence and an editorial annual report, not a browser-rendered PowerPoint, SaaS dashboard, generic wellness landing page, or a grid of identical rounded cards.

Prefer full-screen compositions, asymmetric editorial grids, large authentic photography, strong typography and whitespace, sticky storytelling, restrained scroll transitions, integrated charts, decision architecture, and an alternating rhythm of photographic, analytical, light, and dark sections.

Avoid generic medical stock or AI-generated interiors when official VERBA imagery exists, black-and-gold luxury, decorative gradients, glassmorphism overload, repeated cards, template hero layouts, and excessive animation.

## First implementation checkpoint
Before rebuilding the full review, implement and visually verify only:
1. Cinematic Thesis Hero.
2. Trend Proof for supplement plan performance: January up to 117%, February 34–62%, March 7–13%, benchmark 100%.
3. Q2 Decision System with five priorities shown as an actual hierarchy, not five equal cards.

Preserve accessibility, responsive behavior, and `prefers-reduced-motion`.
